# FloatingHelper

FloatingHelper 是一个适用于 Minecraft `1.21.10` 的 Fabric 客户端模组。

## 功能特性

- 在主界面和游戏内显示 `yc_ui` 悬浮人物
- 在主界面显示跟随 `yc_ui` 的绿色粗体提示文字
- 进入主界面时会随机显示一句，点击 `yc_ui` 会切换到剩余句子
- 提示文字会在约 1 秒内渐显，模拟人物正在说话
- 提示文字默认显示在人物外侧上方，但现在也可以在编辑页中单独拖动位置
- 提示文字大小仍然跟随 `yc_ui` 缩放，不单独调整大小
- 长句会自动换行，不再把第三行内容直接截掉
- 遇到 `，` 或 `？` 时会优先断行，避免单行过长
- 提示文字不再使用阴影，避免部分字被阴影影响清晰度
- 可在 Mod Menu 中切换显示/隐藏，并编辑 `yc_ui` 的位置、大小、镜像，以及提示文字的位置
- 人物位置会随界面尺寸变化保持相对位置
- `yc_ui` 靠左时保持原图朝向，靠右时会自动左右翻转，让人物朝向屏幕内侧

## 环境信息

```properties
minecraft_version=1.21.10
yarn_mappings=1.21.10+build.3
loader_version=0.18.4
fabric_version=0.138.4+1.21.10
gradle=9.2.1
java=21
```

## 构建

```powershell
.\gradlew.bat build
```

构建产物会生成在 `build/libs/` 目录中。

## Mod Menu 配置

安装 `Mod Menu` 后，可以在 `FloatingHelper` 的配置页中使用以下功能：

- `显示 / 隐藏`：切换悬浮人物在主界面和游戏内的显示状态
- `编辑位置与大小`：打开布局编辑界面

说明：

- 红框拖动和缩放 `yc_ui`
- 绿框拖动主界面的提示文字
- 提示文字不会镜像，也不会单独缩放
- 主界面的下一句切换由点击 `yc_ui` 触发

## 默认快捷键

- `X`：显示 / 隐藏悬浮界面
- `Z`：打开 FloatingHelper 配置界面

这些按键可以在游戏的“选项 -> 控制”中重新绑定。
