# FloatingHelper

FloatingHelper 是一个适用于 Minecraft `1.21.10` 的 Fabric 客户端模组。

## 功能特性

- 在主界面和游戏内分别显示独立的 `yc_ui`
- 主界面 UI 和游戏内 UI 现在使用两套独立的位置、大小和镜像配置
- 主界面会显示绿色粗体说话文字，游戏内 UI 不显示文字
- 主界面文字会在约 1 秒内渐显，模拟人物正在说话
- 主界面文字默认显示在人物外侧上方，但也可以在编辑页中单独拖动位置
- 主界面文字大小仍然跟随主界面 `yc_ui` 缩放，不单独调整大小
- 长句会自动换行，不再把第三行内容截掉
- 遇到 `，` 或 `？` 时会优先断行，避免单行过长
- 主界面文字恢复了阴影效果，并保持每一行单独居中
- 可在 Mod Menu 中分别编辑主界面 UI 和游戏内 UI
- `yc_ui` 靠左时保持原图朝向，靠右时会自动左右翻转，让人物朝向屏幕内侧

## 环境信息

```properties
minecraft_version=1.21.10
yarn_mappings=1.21.10+build.3
loader_version=0.18.4
fabric_version=0.138.4+1.21.10
gradle=9.2.1
java=21
```

## 构建

```powershell
.\gradlew.bat build
```

构建产物会生成在 `build/libs/` 目录中。

## Mod Menu 配置

安装 `Mod Menu` 后，可以在 `FloatingHelper` 的配置页中使用以下功能：

- `显示 / 隐藏`：切换主界面和游戏内悬浮人物的显示状态
- `编辑主界面 UI`：编辑主界面人物和主界面文字
- `编辑游戏内 UI`：仅编辑游戏内人物

说明：

- 主界面编辑页中，红框拖动主界面人物，绿框拖动主界面文字
- 游戏内编辑页中，只显示和编辑游戏内 `yc_ui`
- 这次改动只保存在本地，不自动推送到 GitHub

## 默认快捷键

- `X`：显示 / 隐藏悬浮界面
- `Z`：打开 FloatingHelper 配置界面

这些按键可以在游戏的“选项 -> 控制”中重新绑定。
